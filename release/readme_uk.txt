Що таке OmegaT?
===============

OmegaT — це безкоштовна та відкрита мультиплатформенна програма для автоматизованого перекладу, яка підтримує нечіткі збіги, пам’ять перекладів, пошук за ключовими слова, глосарії та використання раніше виконаних перекладів у оновлених проєктах.

Умови для встановлення OmegaT
=============================

Для роботи OmegaT необхідно, щоб у вашій системі було встановлене середовище виконання Java Runtime Environment (JRE) версії 1.8 або пізнішої.

Щоб не витрачати часу на пошук, стягнення та встановлення середовища виконання, рекомендується використовувати інсталяційні пакети OmegaT, які містять JRE.



Встановлення OmegaT (Windows)
=============================

Запустіть програму встановлення.



Встановлення OmegaT (Mac)
=========================

Розпакуйте zip-архів OmegaT, який містить теку із файлом документації та застосунком OmegaT. Перемістіть цю теку до теки програм або в інше місце, звідки ви зможете запустити застосунок.



Встановлення OmegaT (Linux)
===========================

Помістіть архів в будь-яку теку та розпакуйте його — програма OmegaT готова до роботи.

Крім того, для більшої зручності та легкості у користуванні ви може встановити програму скриптом встановлення (linux-install.sh). Для цього запустіть емулятор термінала (консоль), перейдіть у теку, де розміщуються файли OmegaT.jar і linux-install.sh, і запустіть скрипт командою
sudo ./linux-install.sh
або
su -c ./linux-install.sh



Встановлення OmegaT (Solaris, FreeBSD та інші Unix-подібні системи)
===================================================================

Помістіть архів в будь-яку теку та розпакуйте його — програма OmegaT готова до роботи.



Використання OmegaT за допомогою Java Web Start (всі платформи)
===============================================================

Якщо у вас вже встановлено середовище JRE, то найпростіший спосіб встановити OmegaT — це Java Web Start.

Для цього стягніть указаний нижче файл і запустіть його:

   http://omegat.sourceforge.net/webstart/OmegaT.jnlp

В результаті на ваш комп’ютер будуть встановлені компоненти, необхідні для роботи OmegaT, і сама програма. Для подальших запусків не обов’язково бути під’єднаним до мережі.



Запуск OmegaT (Windows)
=======================

Якщо під час установлення ви створили ярлик на робочому столі, то просто двічі клацніть по ньому.

Програма встановлення може створити ярлики в меню «Пуск», на робочому столі та на панелі швидкого запуску. Крім того, можна створити ярлик, перетягнувши файл OmegaT.exe у меню «Пуск», на робочий стіл або на панель швидкого запуску.

Якщо у файловому менеджері (Провіднику Windows) видно лише файл OmegaT, а не OmegaT.exe, то увімкніть показ розширень файлів у налаштуваннях Провідника Windows.



Запуск OmegaT (Mac)
===================

Двічі клацніть по застосунку OmegaT.

Для зручності запуску можна перетягнути застосунок OmegaT в док або на панель Finder. Крім того, OmegaT можна запустити з результатів пошуку Spotlight.



Запуск OmegaT (Linux)
=====================

Якщо програма OmegaT була встановлена скриптом linux-install.sh, то запускати її можна, натиснувши

  Alt+F2

та ввівши

  «omegat» (без лапок).

Для зручнішого запуску OmegaT можна використовувати скрипт на мові Kaptain (omegat.kaptn), який міститься у пакеті встановлення. Для використання цього скрипту спершу встановіть Kaptain. Потім цей Kaptain-скрипт можна запустити, натиснувши

  Alt+F2

та ввівши

  «omegat.kaptn» (без лапок).



Запуск OmegaT з командного рядка (всі системи)
==============================================

Для запуску OmegaT введіть команду:

cd <шлях до теки, де міститься файл OmegaT.jar>

<повний шлях до виконуваного файлу Java> -jar OmegaT.jar

(Виконуваний файл Java — це файл java в GNU/Linux або java.exe у Windows.
Якщо Java встановлена на рівні системи та шлях до виконуваного файлу додано до змінної PATH, то при запуску OmegaT повний шлях до Java вводити не обов’язково.)
